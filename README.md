## 朋也社区主题

### pybbs主题提交说明

1. 请以发pull request的方式提交主题
2. 主题请以文件夹命名好，全称只能是字母、下滑线
3. 静态资源(css, js, image)请放在主题文件夹里
4. 请在主题文件夹里创建`README.md`文件，以说明主题用法，以及注意事项
5. 请注明开源协议
6. 如果想获得他人的捐助，可以在 `README.md` 里贴上自己的收款方式，比如支付宝二维码等
7. 请放上主题页面截图
